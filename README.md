# AI Career Navigator - Path Way To Success

## Overview

AI Career Navigator is an AI-powered career guidance platform that helps students and professionals discover personalized career pathways based on education, interests, aptitude, skills, and career goals.

## Features

* Personalized Career Recommendations
* Aptitude-Based Guidance
* Skill Gap Analysis
* Course Recommendations
* University Suggestions
* Placement Opportunities
* Interactive Dashboard

## Technology Stack

### Frontend

* React.js
* TypeScript
* Tailwind CSS
* Vite

### Backend

* Python
* Flask

### AI/ML

* Recommendation System
* Machine Learning Models
* Natural Language Processing (NLP)

## Workflow

1. User enters educational background.
2. User selects interests and career preferences.
3. System analyzes skills and aptitude.
4. AI model generates personalized career pathways.
5. Recommended courses, colleges, and opportunities are displayed.
